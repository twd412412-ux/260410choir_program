# 광주교회 찬양대 앱 사용 설명서

일반 단원이 스마트폰으로 찬양곡과 일정을 확인할 때 쓰는 기본 안내서입니다.

기능 확인 기준: `main` 커밋 `7700d46912731412befb021ea726bf0ffbdd76df`, 배포 링크 https://twd412412-ux.github.io/260410choir_program/

설명서 작성 기준: 2026년 6월 22일, `USER_GUIDE.md`

## 1. 앱 접속하기

스마트폰에서 아래 링크를 엽니다.

https://twd412412-ux.github.io/260410choir_program/

처음 열면 상단에 `광주교회 찬양대`가 보이고, 하단에는 기본 탭이 보입니다.

- `곡목록`
- `즐찾`
- `홈`
- `일정`

PC나 윈도우에서도 같은 링크로 접속할 수 있습니다. 다만 화면은 스마트폰 사용에 맞춰져 있습니다.

## 2. 홈에서 돌아오는 주일 찬송곡 확인하기

앱을 열면 기본으로 `홈` 화면이 열립니다.

홈 화면에서는 다음 내용을 확인할 수 있습니다.

- 오늘 날짜
- 돌아오는 주일 찬송곡
- 지난 주일 찬송곡
- 그 다음 주일 찬송곡
- 이번 주 일정

찬송곡 제목을 누르면 곡 상세 화면으로 이동합니다.

일정이 있으면 `이번 주 일정` 영역에 날짜, 시간, 장소가 표시됩니다. 일정 항목을 누르면 자세한 내용을 볼 수 있습니다.

## 3. 곡목록 보기

하단의 `곡목록`을 누르면 찬양대 곡 목록을 볼 수 있습니다.

곡목록 화면에는 다음 기능이 있습니다.

- 검색창: `곡명, 단원, 반주자 검색...`
- 정렬: `최신순`, `오래된순`, `곡명 가나다순`, `곡명 역순`, `많이 부른 곡`, `오래 안 부른 곡`
- 연도 필터
- 분류 필터
- 요일 필터

곡 카드에는 곡명, 날짜, 분류, 참여단원, 반주자 정보가 표시됩니다.

## 4. 곡 검색하기

`곡목록` 화면 상단 검색창에 찾고 싶은 내용을 입력합니다.

검색할 수 있는 내용은 다음과 같습니다.

- 곡명
- 참여단원
- 반주자
- 분류
- 장소

검색어를 입력하면 잠시 후 목록이 자동으로 좁혀집니다.

띄어쓰기나 일부 기호가 달라도 최대한 검색되도록 처리되어 있습니다.

## 5. 곡 상세 보기

곡목록이나 홈 화면에서 곡 제목을 누르면 곡 상세 화면이 열립니다.

곡 상세 화면에서는 다음 내용을 볼 수 있습니다.

- 날짜
- 카테고리
- 세부분류
- 장소
- 참여단원
- 반주자

YouTube 링크가 등록된 곡은 `YouTube에서 듣기` 버튼이 보입니다.

YouTube 링크가 없는 곡은 `YouTube에서 검색` 버튼이 보입니다.

하단의 `공유` 버튼을 누르면 곡 정보를 다른 사람에게 보낼 수 있습니다. 공유가 지원되지 않는 환경에서는 클립보드에 복사됩니다.

## 6. 즐겨찾기 추가와 삭제

자주 보는 곡은 즐겨찾기에 추가할 수 있습니다.

추가 방법:

1. `곡목록`에서 원하는 곡을 찾습니다.
2. 곡 카드 오른쪽의 별표를 누릅니다.
3. 별표가 채워지면 즐겨찾기에 추가된 상태입니다.

삭제 방법:

1. 하단의 `즐찾`을 누릅니다.
2. 삭제할 곡의 별표를 다시 누릅니다.
3. 즐겨찾기에서 제거됩니다.

곡 상세 화면에서도 `즐겨찾기` 버튼으로 추가하거나 삭제할 수 있습니다.

## 7. 로그인과 PIN 입력

상단의 사람 아이콘을 누르면 `내 프로필` 화면이 열립니다.

로그인 화면에는 다음 입력칸이 있습니다.

- 이름
- PIN (4자리)

이름과 PIN을 입력한 뒤 `로그인`을 누르면 내 계정으로 접속됩니다.

로그인하지 않아도 모든 곡은 열람할 수 있습니다. 앱 화면에도 `로그인 없이도 모든 곡 열람 가능`이라고 안내됩니다.

로그인이 필요한 경우는 주로 다음과 같습니다.

- 즐겨찾기를 다른 기기에서도 이어서 쓰고 싶을 때
- 아카이브에서 공감하기를 누를 때
- 권한이 있는 단원이 권한자 전용 기능을 사용할 때

PIN을 모르면 총무부나 담당자에게 문의합니다.

## 8. 즐겨찾기 동기화

로그인하지 않은 상태에서도 즐겨찾기는 현재 기기에 저장됩니다.

로그인하면 즐겨찾기가 내 계정에 저장되어 다른 기기에서도 이어서 사용할 수 있습니다.

이미 로그인 계정에 저장된 즐겨찾기가 있으면, 로그인할 때 그 목록을 불러옵니다.

## 9. 일정 확인하기

하단의 `일정`을 누르면 월별 캘린더가 열립니다.

일정 화면에서는 다음 기능을 사용할 수 있습니다.

- 이전 달, 다음 달 이동
- 보기 방식 선택: `기본 · 점+밴드`, `스택형`
- 날짜 선택
- 선택한 날짜의 일정 확인
- 선택한 날짜의 찬양곡 확인

날짜를 누르면 아래쪽에 해당 날짜의 일정이 표시됩니다.

일정 항목을 누르면 일정 상세 화면이 열립니다.

일정 상세 화면에서는 다음 내용을 볼 수 있습니다.

- 날짜
- 시간
- 장소
- 연습곡
- 메모
- 프로그램 브리핑이 있는 경우 프로그램, 당일 단복, 추가 안내

## 10. 아카이브 보기와 공감하기

`아카이브`는 기본 일반 화면에서는 보이지 않을 수 있습니다.

아카이브 탭은 로그인했거나 아카이브 권한이 있는 경우에 보입니다.

아카이브에서는 파트별 사진 자료를 볼 수 있습니다.

보기 설정에서 다음 항목을 바꿀 수 있습니다.

- 파트
- 유형
- 보기 방식: `2열`, `3열`, `1열 (크게)`

사진 자료에는 `공감하기` 버튼이 있습니다.

공감하기는 로그인 후 사용할 수 있습니다. 로그인하지 않은 상태에서 누르면 로그인 안내가 표시됩니다.

사진 올리기, 수정, 삭제는 권한자 전용 기능입니다.

## 11. 하단 탭 구조

일반 단원이 주로 쓰는 하단 탭은 다음과 같습니다.

| 탭 | 용도 |
| --- | --- |
| `곡목록` | 전체 곡을 보고 검색합니다. |
| `즐찾` | 즐겨찾기한 곡을 모아 봅니다. |
| `홈` | 돌아오는 주일 찬송곡과 이번 주 일정을 봅니다. |
| `일정` | 월별 일정과 날짜별 찬양곡을 확인합니다. |

권한이 있는 경우에는 `아카이브`, `통계`, `관리` 같은 탭이 추가로 보일 수 있습니다.

이 탭들은 권한자 전용 기능이므로 일반 단원 설명서에서는 자세히 다루지 않습니다.

## 12. 상단 버튼 사용하기

상단 오른쪽에는 아이콘 버튼이 있습니다.

| 버튼 | 용도 |
| --- | --- |
| 사람 아이콘 | 내 프로필, 로그인, 로그아웃 |
| 달 아이콘 | 다크모드 켜기와 끄기 |
| 자물쇠 아이콘 | 권한자 로그인 또는 권한자 모드 |
| 새로고침 아이콘 | 최신 곡, 일정, 공지 정보를 다시 불러오기 |

일반 단원은 보통 사람 아이콘과 새로고침 아이콘을 주로 사용합니다.

## 13. 새로고침하기

최신 곡이나 일정이 바로 보이지 않으면 상단의 새로고침 아이콘을 누릅니다.

새로고침을 누르면 `새로고침 중...` 안내가 나온 뒤, 곡과 일정 정보가 다시 불러와집니다.

완료되면 `최신 데이터로 업데이트 완료!` 안내가 표시됩니다.

## 14. 스마트폰에서 앱처럼 쓰기

앱을 지원하는 브라우저에서는 `홈 화면에 추가` 안내가 뜰 수 있습니다.

안내가 보이면 `설치`를 누르면 됩니다.

설치하면 스마트폰 홈 화면에서 앱처럼 빠르게 열 수 있습니다.

현재 앱은 홈 화면 추가용 설정은 있지만, 오프라인 전용 앱으로 안내하지는 않습니다. 최신 정보 확인을 위해 인터넷 연결 상태에서 사용하는 것이 좋습니다.
