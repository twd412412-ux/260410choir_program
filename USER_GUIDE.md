# 광주교회 찬양대 앱 v5 일반 단원 안내서

일반 단원이 스마트폰으로 찬양곡, 일정, 악보, 자리배치를 빠르게 확인하기 위한 안내서입니다.

기능 확인 기준: 2026년 7월 5일 GitHub `main` 최신 커밋  
앱 링크: https://twd412412-ux.github.io/260410choir_program/

## 1. 먼저 이것만 기억하세요

- 앱은 스마트폰 화면 기준으로 만들어져 있습니다.
- 로그인하지 않아도 기본 곡 목록과 일정은 볼 수 있습니다.
- 내 즐겨찾기를 여러 기기에서 이어 쓰거나, 내 자리배치를 자동으로 찾으려면 로그인하는 것이 좋습니다.
- 최신 내용이 바로 안 보이면 홈 화면 아래쪽의 `앱 업데이트` 또는 상단 새로고침 아이콘을 누릅니다.

## 2. 접속과 로그인

1. 스마트폰에서 앱 링크를 엽니다.
2. 상단 사람 아이콘을 누릅니다.
3. 이름과 4자리 PIN을 입력합니다.
4. `로그인`을 누릅니다.

PIN을 모르면 총무부나 담당자에게 문의합니다.

## 3. 홈 화면에서 바로 확인하기

앱을 열면 홈 화면에서 이번 주에 필요한 내용을 먼저 볼 수 있습니다.

- 돌아오는 주일 찬송곡
- 지난 주일 찬송곡
- 그 다음 주일 찬송곡
- 이번 주 일정
- 공개된 찬양의밤 자리배치
- 이번 주 악보 바로가기

곡 제목을 누르면 곡 상세 화면으로 이동합니다. 일정 항목을 누르면 일정 상세를 볼 수 있습니다.

## 4. 악보 다운로드

v5에서 일반 단원이 가장 많이 쓰는 기능입니다.

1. 하단의 `악보` 탭을 누릅니다.
2. 상단 검색창에 곡명이나 버전을 입력합니다.
3. 원하는 악보 카드의 `열기` 버튼을 누릅니다.
4. PDF가 열리면 휴대폰의 공유 또는 저장 기능으로 내려받습니다.

알아둘 점:

- 공개된 최신 악보만 표시됩니다.
- PDF는 `열기`를 눌렀을 때만 불러옵니다.
- 홈 화면의 이번 주 악보 영역에서도 바로 열 수 있습니다.
- 악보가 보이지 않으면 아직 공개되지 않았거나 등록된 파일이 없는 상태입니다.

## 5. 찬양의밤 자리배치 확인

자리배치가 공개되면 홈 화면에 `찬양의밤 자리배치` 카드가 표시됩니다.

확인 방법:

1. 홈 화면에서 `찬양의밤 자리배치` 카드의 `보기`를 누릅니다.
2. 로그인되어 있으면 내 이름의 자리가 자동으로 강조됩니다.
3. 로그인하지 않았거나 다른 사람 자리를 확인하려면 검색창에 이름이나 파트를 입력합니다.
4. 확대/축소 버튼으로 보기 크기를 조절합니다.

알아둘 점:

- 공개된 자리배치가 없으면 카드가 보이지 않거나 `아직 공개된 자리배치가 없습니다`로 표시됩니다.
- 자리배치 화면의 `앞쪽 / 무대 아래`, `지휘` 표시를 기준으로 실제 위치를 확인합니다.
- 이름이 길면 칸 안에서 줄어 보일 수 있으니 검색으로 한 번 더 확인합니다.

## 6. 곡목록과 검색

하단의 `곡목록`에서 전체 찬양곡을 찾을 수 있습니다.

검색할 수 있는 내용:

- 곡명
- 참여 단원
- 반주자
- 분류
- 장소

필터와 정렬:

- 최신순
- 오래된순
- 곡명 가나다순
- 많이 부른 곡
- 오래 안 부른 곡
- 연도, 분류, 요일 필터

검색 조건을 많이 걸었을 때는 결과 수 옆의 `초기화`를 누르면 처음 상태로 돌아갑니다.

## 7. 곡 상세와 YouTube

곡목록이나 홈에서 곡 제목을 누르면 곡 상세 화면이 열립니다.

확인할 수 있는 내용:

- 날짜
- 분류
- 장소
- 참여 단원
- 반주자
- YouTube 링크 또는 YouTube 검색 버튼

하단의 `공유` 버튼을 누르면 곡 정보를 다른 사람에게 보낼 수 있습니다.

## 8. 즐겨찾기

자주 보는 곡은 별표로 즐겨찾기할 수 있습니다.

추가:

1. `곡목록`에서 곡을 찾습니다.
2. 곡 카드의 별표를 누릅니다.
3. 별표가 채워지면 저장된 상태입니다.

확인:

1. 하단의 `즐찾`을 누릅니다.
2. 즐겨찾기한 곡과 최근 본 곡을 확인합니다.

로그인하면 즐겨찾기를 다른 기기에서도 이어서 사용할 수 있습니다.

## 9. 일정 확인

하단의 `일정`을 누르면 월별 캘린더가 열립니다.

사용 방법:

1. 이전 달, 다음 달 버튼으로 월을 이동합니다.
2. 원하는 날짜를 누릅니다.
3. 아래쪽에서 그날의 일정과 찬양곡을 확인합니다.
4. 일정 항목을 누르면 시간, 장소, 연습곡, 메모를 볼 수 있습니다.

보기 방식은 `기본 · 점+밴드` 또는 `스택형` 중 편한 방식으로 바꿀 수 있습니다.

## 10. 하단 탭 빠른 정리

| 탭 | 일반 단원 사용 목적 |
| --- | --- |
| `곡목록` | 전체 곡 검색 |
| `즐찾` | 자주 보는 곡 모아보기 |
| `홈` | 이번 주 곡, 일정, 자리배치, 악보 빠른 확인 |
| `악보` | 공개된 최신 악보 PDF 열기 |
| `일정` | 월별 일정과 날짜별 찬양곡 확인 |

권한이 있는 단원에게는 관리자용 탭이나 기능이 추가로 보일 수 있습니다. 일반 단원은 위 탭만 사용하면 됩니다.

## 11. 문제가 있을 때

- 최신 악보나 일정이 안 보이면 새로고침 또는 `앱 업데이트`를 누릅니다.
- 악보가 열리지 않으면 인터넷 연결을 확인합니다.
- 자리배치에 내 이름이 없으면 이름 검색을 먼저 해보고, 그래도 없으면 파트 담당자나 총무부에 확인합니다.
- PIN을 모르면 총무부나 담당자에게 문의합니다.
